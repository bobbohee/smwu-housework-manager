우리집 살림 매니저 — 확정 설계 요약

브레인스토밍에서 결정된 사항을 정리했습니다. 터미널에서 피드백 주세요.

🏠 그룹 & 인증

  • • Firebase Auth: 이메일/비밀번호만 저장
  • • 이름은 그룹 가입 시 입력 (그룹별 다른 이름 가능)
  • • 다중 그룹 지원 — 홈 상단에서 그룹 전환
  • • 초대 코드: 제한 없이 언제든 사용 가능
  • • 그룹 생성자 = 방장 (강퇴 권한)

📋 집안일 모드

순번제

  • 완료 버튼으로 처리
  • 1번 완료 = 차례 1번 소진
  • 바로 다음 사람으로 넘어감
  • choreLog에 기록됨

요일 고정제

  • 완료 버튼 없음
  • 해당 요일에 알림만 표시
  • "오늘 쓰레기 배출: 보희"
  • 기록 저장 안 함

💣 꽝뽑기 (랜덤 돌리기)

  • • 그룹 멤버 전원 표시 (체크박스로 참여/제외)
  • • 당첨 인원 수 조정 가능 (기본 1명)
  • • 폭탄 아이콘 + 당첨자 이름 표시
  • • 결과는 choreLog에 기록

📅 캘린더 & 통계

캘린더

  • 순번제 완료 기록만 표시
  • 집안일별 색상 구분
  • "설거지: 보희" 형태
  • 클릭 시 상세 팝업

통계

  • 숫자 + 차트 둘 다
  • 멤버별 완료 횟수
  • 집안일별 완료 횟수
  • 이번 달 기준

⚙️ 아키텍처

  • • 프론트엔드 중심 — Firestore 직접 읽기/쓰기
  • • API Route는 최소한만 (서버 검증 필요 시)
  • • Firestore Security Rules로 권한 제어
  • • 집안일 등록 시 팔레트에서 색상 선택
